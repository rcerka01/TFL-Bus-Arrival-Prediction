The development of this application has purely academic matter. The main aim is to build a reactive web application by using Play framework and Scala as a programming language.
Target application will be connecting via WebSockets to the TFL produced a data stream which will be handled by the WS connection library and reproduced via Iterattor and Enumerator classes (the cornerstone of Scala reactive programming paradigm).
The overall aim is to test stability and actual performance of this architecture.
